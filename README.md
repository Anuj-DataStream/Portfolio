### `PROJECT 02` — EcoPal — Smart Plant Automation System
> **IoT · Embedded Systems · C++** · 2025

<table>
<tr>
<td width="60%">

Designed and built a **sensor-driven plant automation system** that monitors soil conditions and controls irrigation in real time — reducing manual intervention through reliable, low-cost automation.

**How it works:**
- 🌱 Capacitive soil moisture sensor continuously monitors soil state
- 💧 Arduino rotates servo motor 90° to open/close the water valve
- 🟢 Green LED = Soil wet, valve closed · 🔴 Red LED = Soil dry, valve open
- Supports both **single and multi-plant** configurations with independent sensor channels
- Fully autonomous cycle — no human input required during operation

**Objectives achieved:**
- ✅ Real-time soil moisture monitoring
- ✅ Servo-controlled automatic irrigation
- ✅ LED status indicators
- ✅ Single & multi-plant configurations

</td>
<td width="40%" align="center">

```cpp
// EcoPal Logic (simplified)
void loop() {
  int moisture = analogRead(SENSOR);

  if (moisture < DRY_THRESHOLD) {
    // Dry → Open valve
    servo.write(0);
    digitalWrite(RED_LED, HIGH);
    digitalWrite(GREEN_LED, LOW);
  } else {
    // Wet → Close valve
    servo.write(90);
    digitalWrite(GREEN_LED, HIGH);
    digitalWrite(RED_LED, LOW);
  }
}
```

</td>
