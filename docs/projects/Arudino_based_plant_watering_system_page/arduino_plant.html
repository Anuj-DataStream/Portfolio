The Project

Project Overview

🎯
Objective
To build a fully automatic watering system that monitors soil moisture, controls water flow using a servo motor, and indicates status via Green & Red LEDs — all without human intervention.
💡
Inspiration
The idea came from exploring YouTube automation videos and a deep interest in Arduino electronics. Developed as part of the Entrepreneur Mindset Curriculum (EMC) at RPVV SEC 19.
🌱
Problem Solved
Ensuring plants are watered correctly — even when we're not home. Over-watering and under-watering are both harmful. This system achieves precise, automatic irrigation based on real soil data.
📈
Skills Gained
Hands-on experience in circuit design, C++ programming, sensor integration, IoT architecture, and end-to-end project management — from concept to working prototype.

Working Principle

📡

Arduino Decides
Code logic compares reading against moisture threshold

⚙️
Servo Rotates

→
🔴🟢
LED Status
Coloured LEDs visually indicate current watering state
🟢 Moist Soil — Water OFF
Moist soil conducts electricity. Sensor triggers Arduino to rotate servo 90°, shutting the valve. Green LED turns ON.
🔴 Dry Soil — Water ON
Dry soil doesn't conduct. Sensor triggers Arduino to open the valve fully. Red LED turns ON until soil is moist.

Project Report

Anuj Kumar RPVV SEC 19 EMC Program 2025 Arduino + IoT
📘 Introduction
This project aims to solve a real-world problem — ensuring plants are watered even when we are not home. I developed it as part of the EMC program, inspired by online automation videos and my growing interest in Arduino and electronics.
🎯 Objectives
  • Monitor soil moisture using a capacitive sensor
  • Control water flow automatically via a servo-operated valve
  • Indicate watering status clearly using coloured LEDs
  • Support both single and multi-plant configurations
⚙️ Working Principle
Moist soil conducts electricity — the sensor detects it and Arduino rotates the servo 90° to shut the water supply (Green LED ON). Dry soil doesn't conduct — the sensor triggers Arduino to open the valve (Red LED ON). The cycle repeats continuously and fully autonomously.
📚 Inspiration & Learning
The idea originated from YouTube automation channels. I explored Arduino deeply, learned how to write sensor-driven code, design circuits, and integrate electronics practically. The EMC curriculum provided the framework to bring this idea to a functional prototype.
✅ Conclusion
A simple yet powerful solution for plant care, combining physics and technology. This project significantly boosted my skills in innovation, coding, circuit design, and independent project execution — and proved that real-world problems can be solved with accessible technology.
